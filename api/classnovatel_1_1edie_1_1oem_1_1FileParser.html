  • stMetaData_ – [out] A reference to a MetaDataStruct to be populated by the FileParser.

Returns:

An error code describing the result of parsing. SUCCESS: A message was framed, decoded, filtered, encoded and stored internally, pointed to by stMessageData_. UNKNOWN: A message could not be found and unknown bytes were returned if requested in the ParserConfigStruct given to SetConfig(). FILE_EMPTY: There are no more bytes to parse in the file provided.
