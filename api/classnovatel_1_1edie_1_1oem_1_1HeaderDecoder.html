STATUS Decode(unsigned char *pucLogBuf_, IntermediateHeader &stInterHeader_, MetaDataStruct &stMetaData_) const

Decode an OEM message header from the provided frame.

Parameters:

  • pucLogBuf_ – [in] A pointer to an OEM message header.

  • stInterHeader_ – [out] The IntermediateHeader to be populated.

  • stMetaData_ – [inout] MetaDataStruct to provide information about the frame and be fully populated to help describe the decoded log.

Returns:

An error code describing the result of decoding. SUCCESS: The operation was successful. NULL_PROVIDED: pucHeader_ is a null pointer. NO_DATABASE: No database was ever loaded into this component. FAILURE: Failed to decode a header field. UNSUPPORTED: Attempted to decode an unsupported format. UNKNOWN: The header format provided is not known.
