STATUS GetFrame(unsigned char *pucFrameBuffer_, uint32_t uiFrameBufferSize_, MetaDataStruct &stMetaData_)

Frame an OEM message from bytes written to the Framer.

Parameters:

  • pucFrameBuffer_ – [out] The buffer which the Framer should copy the framed OEM message to.

  • uiFrameBufferSize_ – [in] The length of pucFrameBuffer_.

  • stMetaData_ – [out] A MetaDataStruct to contain some information about OEM message frame.

Returns:

An error code describing the result of framing. SUCCESS: An OEM message frame was found. NULL_PROVIDED: pucFrameBuffer_ is a null pointer. UNKNOWN: The bytes returned are unknown. INCOMPLETE: The framer found what could be an OEM message, but there are no more bytes in the internal buffer. BUFFER_EMPTY: There are no more bytes in the internal buffer. BUFFER_FULL: pucFrameBuffer_ has no more room for added bytes, according to the size specified by uiFrameBufferSize_.
