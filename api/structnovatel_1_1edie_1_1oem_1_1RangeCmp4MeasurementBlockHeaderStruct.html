struct RangeCmp4MeasurementBlockHeaderStruct

Measurement Block Header structure to contain the values within the compressed bitfields for OEM4 RANGECMP4 messages.

NOTE: None of the RANGECMP4 structures are packed because the method of decompression is not a cast-based decode operation. The bitfields are far too complex and oddly-sized such that each field must be extracted from the binary data buffer one at a time. A bitfield helper function should be used to determine and populate the following structures. These are for containment purposes only.
