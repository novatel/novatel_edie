class FileStream

A class will provide file operation handlers. More detailed FileStream class description. It will also support file names with wide characters.

Public Types

enum class FILE_MODES

An enum.

File Operations Enumeration.

Values:

enumerator APPEND

Append may refer to the process of attaching or combining data with another file or set of data.

enumerator INSERT

Insert Data into the file

enumerator INPUT

Open file as Input for read

enumerator OUTPUT

Open file as output for write into the file

enumerator TRUNCATE

Truncate some part of data in file
