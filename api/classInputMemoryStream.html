class InputMemoryStream : public InputStreamInterface

A Derived Class for writing buffer data to circullar buffer which will be used for decoding by decoder.

Class will provide methods to update circullar buffer which will be used by decoder for decoding data. Charater buffer data could be Ascii/Binary type.
