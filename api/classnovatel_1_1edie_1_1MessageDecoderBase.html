STATUS Decode(unsigned char *pucMessage_, std::vector<FieldContainer> &stInterMessage_, MetaDataBase &stMetaData_) const

Decode a message body from the provided frame.

Remark

Note, that pucMessage_ must not point to the message header, rather the message body. This can be done by advancing the pointer of a message frame by stMetaData.uiHeaderLength.

Parameters:

  • pucMessage_ – [in] A pointer to a message body.

  • stInterMessage_ – [out] The IntermediateLog to be populated.

  • stMetaData_ – [inout] MetaDataStruct to provide information about the frame and be fully populated to help describe the decoded log.

Returns:

An error code describing the result of decoding. SUCCESS: The operation was successful. NULL_PROVIDED: pucMessage_ is a null pointer. NO_DATABASE: No database was ever loaded into this component. NO_DEFINITION: No definition for the current log was found in the provided database. FAILURE: Failed to decode a header field. UNSUPPORTED: Attempted to decode an unsupported format. UNKNOWN: The header format provided is not known.
